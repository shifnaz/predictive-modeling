# Predictive Modeling: Regression & Classification (2021)

## 📌 Project Overview
This project focuses on the practical application of regression techniques to solve real-world business problems. It covers the end-to-end process of building, validating, and interpreting predictive models.

## 📊 Business Problems Solved
1. **Linear Regression:** - Built a model to predict a continuous numerical output (e.g., house prices or sales volume).
   - Focused on identifying the "drivers" of the target variable.
2. **Logistic Regression:** - Developed a classification model to predict binary outcomes (e.g., Customer Churn: Yes/No).
   - Used probability thresholds to categorize data points.

## 🛠️ Technical Workflow
* **EDA & Preprocessing:** Handled missing values, outliers, and performed feature scaling.
* **Multicollinearity Check:** Used **VIF (Variance Inflation Factor)** to ensure independent variables weren't too highly correlated with each other.
* **Model Evaluation:** * For Linear: Analyzed **R-Squared**, **RMSE**, and **Residual Plots**.
    * For Logistic: Evaluated **Confusion Matrix**, **Precision**, **Recall**, and **ROC-AUC curves**.

## 💡 2026 Perspective
Looking back at this project, I see how my documentation standards have matured:
* **Code Commenting:** Today, I would include detailed comments within the code blocks to explain the logic of specific functions and the interpretation of the statistical coefficients, making the model-building process more transparent for stakeholders.
